Creates an easy to use post ticker that slides through the selected posts via shortcode and widget area.

== Description ==
WP Post Ticker plugin is designed to beautifully display your WordPress posts via shortcode and widget area. This WordPress Post Ticker plugin is especially useful when you want to display multiple post categories with different sets of options. You will fall in love with how flexible the options are, see <a href="https://icanwp.com/plugins/wp-post-ticker-pro-demo/">live examples here</a>

This plugin was also created with modern web design best practices in mind. Compatible with all major browsers and most WordPress Themes, WP Post Ticker is responsive and will give your website that modern touch.

== Installation ==
This section describes how to install the plugin and get it working.

1. Upload 'wp-post-ticker' directory to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Press "Add New Post Ticker" to create a ticker and start customizing your settings.
4. You can find your new post ticker in the widget area, or simply copy the shortcode into any page or post.

== Uninstall ==

* All global options will be deleted from the database. Use disable plugin if you want to save your global options.

== Frequently Asked Questions ==

1. Can I display multiple sliding post tickers on the same page?
Yep, you bet!

2. Is the style of the ticker customizable?
Absolutely. You have control over almost every option. Change the colors, width, opacity, size, etc.

3. Is the post ticker responsive and does it work on a mobile setting??
Yes and yes!

4. How flexible are the display settings?
Very flexible. Choose to display/hide private posts, thumbnails, post meta info, categories, and much more.

5. Is this post ticker compatible with other major plugins?
Yes, this the WP Post Ticker is compatible with the following major plugins:
* WooCommerce
* Gravity Forms
* WPTouch Pro
* Jetpack
* Revolution Slider
* Many more..
